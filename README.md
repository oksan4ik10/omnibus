# Omnibus

**Omnibus** — интерактивное веб-приложение, разработанное на **React + TypeScript** с использованием **Vite**. Проект демонстрирует современный подход к разработке SPA, работу с состоянием, формами и адаптивным пользовательским интерфейсом.
 **Демо:** https://omnibus-game.netlify.app/

---

## Стек:
- React 18
- TypeScript
- Vite
- Redux Toolkit
- React Hook Form
- Sass (SCSS)
- Swiper
- ESLint

---

## Функциональность
- Адаптивный и отзывчивый интерфейс  
- Централизованное управление состоянием (Redux Toolkit)  
- Работа с формами и пользовательским вводом  
- Слайдеры и интерактивные элементы  
- Подключение аналитики (Yandex Metrika)

---

## Назначение

Интерактивное приложение по заказку компаний **Сбер**, **Билайн**, **Балтика** и других, демонстрирующее разнообразие карьерных траекторий и возможности профессионального развития в разных направлениях.
